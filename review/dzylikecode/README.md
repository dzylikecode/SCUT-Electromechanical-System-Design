当高速轴带动波发生器凸轮和柔性轴承逆时针连续转动时，柔轮上原来与刚轮啮合的齿对开始啮出后脱开，再转入啮入，然后重新啮合，这样柔轮就相对于刚轮沿着与波发生器相反的方向低速旋转，通过低速轴输出运动。若将柔轮 2 固定，由刚轮 5 输出运动，其工作原理完全相同，只是刚轮的转向将与波发生器的转向相同。
